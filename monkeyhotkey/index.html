몽키 핫키
모두의 핫키 - Simple Hotkey Launcher for Windows

소개

몽키 핫키는 브라우저, 메신저, 이메일 클라이언트와 같이 자주 사용하는 프로그램 및 빈번하게 편집하는 파일 또는 즐겨찾는 폴더등을 단축키를 이용해 쉽고 빠르게 실행시키는 프로그램입니다.

기능

  • Windows XP / Vista / 7 지원
  • 자주 사용하는 프로그램 실행
  • 파일, 폴더 열기
  • 영문, 숫자, 펑션(F1~F2)키와 조합키를 이용한(Ctrl, Alt, Shift) 핫키 관리

최신 버전 내려받기

  • 인스톨러 버전 내려받기
  • 무설치 버전 내려받기
