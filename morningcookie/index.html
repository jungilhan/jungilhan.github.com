모닝 쿠키
소셜 기능을 활용한 TTS 알람 - Prototype - 2012 Facebook Mobile Hack Seoul

소개

모닝 쿠키는 즐겨보는 뉴스, 날씨 정보, RSS 피드 그리고 SNS 및 그 외 관심있는 정보를 알람으로 제공하는 오픈소스 프로젝트입니다. 일반적으로 알람 앱에서 많이 사용하는 음악이나 라디오가 아닌 매일 매일 새로운 정보들을 들으면서 하루를 시작해보는 것은 어떨까요? 다른 사람과 공유하고 싶은 정보가 있다면 '좋아요!' 라고 외쳐보세요.

기능

  • 뉴스, 날씨, 즐겨찾는 RSS 피드를 TTS 엔진을 활용해 알람 제공

미래의 작업 목록

  • SNS 새소식 읽어오기
  • 캘린더 정보 읽어오기
  • 아이폰 버전 만들기
