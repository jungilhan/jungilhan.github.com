Goconut
Editor for Go programming language

소개

Goconut은 Go 프로그래밍 언어 편집기를 만들기 위한 프로젝트입니다. 기본적인 편집 기능, 구문 강조, 자동 완성, 빌드 및 실행이 가능한 편집기를 만드는 것이 최종 목표입니다. 2012 GDG 수원 안드로이드 해커톤에서는 안드로이드에서 동작하는 프로토타입 수준의 Go 언어 편집기를 만들어 볼 계획입니다.

2012 GDG 수원 안드로이드 해커톤에서의 목표

  • 기본적인 편집 기능
  • 구문 강조 (Syntax highlighting)
  • 빌드 & 실행
  • 안드로이드 기기에서 작성한 소스 코드를 클라우드 저장소에 저장하기
