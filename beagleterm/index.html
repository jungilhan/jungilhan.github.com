Beagle Term
Google Chrome based SSH Terminal

소개

Beagle Term은 크롬 확장 기술과 오픈소스 라이브러리를 사용해 크롬 브라우저 기반의 SSH 클라이언트를 만들기 위한 프로젝트입니다.

프로토타입 시연 영상

2011 Google Developers Hackathon 현장에서 개발 후 촬영한 시연 영상입니다.

해커톤에서의 목표

  • 오픈소스 라이브러리인 libssh를 이용해 NPAPI 플러그인 만들기
  • 그럴싸한 터미널 갖추기
  • 크롬 확장 만들기
    • 크롬 앱으로 패키징하기
    • URL 입력 창에 SSH 프로토콜 입력 시 터미널 동작시키기 - Omnibox?
  • 간단한 위젯 만들기
  • 많은 개발자 분들 만나기 :)

참여해주세요

  • Chrome Extensions
  • HTML5/CSS/Javascript
  • SSH
  • VT100 터미널 제어
  • C/C++

  • NaCl(미래의 작업 목록 참고)
  • GAE(미래의 작업 목록 참고)

미래의 작업 목록

  • VT100 터미널 코드와 호환
  • NPAPI 플러그인을 크롬의 NaCl(Native Client)로 대체
  • 개인화 정보 서버와 연동하기 - 클라우드 - GAE 사용하면 될까요?
  • 사용자 편의 기능 추가
  • 다양한 위젯 추가
